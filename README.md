# Customer Spending Behaviour Analysis

## Description
This project analyzes customer spending behavior using data analysis techniques. It helps in understanding how customers spend based on different factors and identifies useful patterns.

## Tools & Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Google Colab

## Features
- Analyzed customer spending patterns
- Identified trends in purchases
- Performed data cleaning and preprocessing
- Visualized data using graphs
